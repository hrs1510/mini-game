# Mini Grid Game

## Overview
The Mini Grid Game is a fun and interactive web-based game where players click on vibrant colored squares that disappear upon clicking. The objective is to click as many squares as possible within a 60-second timer.

## Features
- Vibrant colored grid squares that disappear when clicked.
- A start button to initiate the game.
- A timer that counts down from 60 seconds.
- Simple and engaging gameplay suitable for all ages.

## Files
- **index.html**: Contains the HTML structure of the game, including the grid and timer display.
- **styles.css**: Provides the styling for the game, ensuring a colorful and appealing user interface.
- **script.js**: Implements the game logic, including the timer and click events.

## How to Run the Game
1. Clone the repository or download the files.
2. Open `index.html` in a web browser.
3. Click the "Start" button to begin the game.
4. Click on the colored squares as they appear to score points.
5. Watch the timer countdown and try to click as many squares as possible before time runs out!
